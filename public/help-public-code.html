How to get your public code (address)

  1. Open Atomic Wallet.
  2. Select the coin (Bitcoin / Ethereum / Dogecoin).
  3. Tap Receive.
  4. Tap Copy to copy your public address.
  5. Paste it back on the withdrawal page into Public code.

Safe to share. Never share your private key or seed phrase.
